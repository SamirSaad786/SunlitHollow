# Sunlit Hollow

> You are the Warden of a mystical forest that exists between worlds. Tend it carefully.

**Sunlit Hollow** is a cozy idle game about growing a living forest — one tap at a time — until fireflies drift, a brook babbles, and the aurora crowns the canopy.

## What It Is

A browser-based idle / clicker game where players tap sunlight to grow eight tiers of forest, from seedlings to full aurora bloom. As the forest grows, its soundscape builds procedurally — fireflies appear, a brook unlocks, birdsong layers in. Rebirth resets your progress for permanent Moonbeam bonuses that make the next run richer.

This is more A Short Hike than Cookie Clicker in tone. The goal is an atmosphere, not a number.

## Features

- **Eight forest growth tiers** — from seedling to aurora canopy
- **Procedural ambient soundscape** built with Web Audio API — the forest sounds different as it grows
- **Rebirth system** — reset for permanent Moonbeam bonuses
- **Idle mechanics** — progress continues while you're away
- Single-file HTML — no install, no account, no data collection

## Genre & Tone

Idle / Clicker / Forest Fantasy  
Cozy / Atmospheric / Meditative
